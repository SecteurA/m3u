# M3U Category Editor

A Streamlit web app to upload, view, filter, and download M3U playlists by category.

## Features
- Upload M3U files
- View and select categories/channels
- Remove or add categories to download list
- Download filtered M3U file

## Usage
1. Install requirements:
   pip install -r requirements.txt
2. Run the app:
   streamlit run app.py
